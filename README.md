# Aircraft Schedule

## How to start
How to start
git clone https://github.com/flaviagfigueiredo/aircraft-scheduling.git

* cd aircraft-scheduling
* yarn
* yarn start

Runs the app in the development mode.
Open http://localhost:3000 to view it in the browser.
